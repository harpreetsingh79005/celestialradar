# Project Zenith 🌌

Project Zenith is a modern celestial radar experience built to help explorers find the best view of the sky from their current location. It combines astronomy intelligence, interactive dashboards, and an AI copilot to make every stargazing session feel like mission control.

---

## ✨ What’s Included

- 🛰 **Zenith Radar**: live sky scanning with local horizon tracking
- 🌙 **Moon & Sun panels**: current lunar phase, solar activity, and science insights
- 🛰 **Satellite & orbital event explorer**
- 🌌 **Aurora Lab**: polar light visibility and aurora intelligence
- 🤖 **Nova AI Copilot**: ask astronomy questions with conversational suggestions
- 🧠 **Cosmic Quiz**: interactive astronomy knowledge checks
- 📰 **Space News & mission briefings**
- 🎨 **Modern responsive UI** with immersive night-sky styling

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js for 3D visualizations
- Web Speech API for voice interaction
- Express / Node.js for local server support

---

## 🚀 Run Locally

1. Open the project folder in a terminal.
2. Install dependencies if needed:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```
4. Open your browser to:
   ```
   http://localhost:3000
   ```

## 🧭 Notes

- Nova is the AI assistant designed to help you explore the sky.
